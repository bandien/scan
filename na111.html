AT Command - Chan Doan Qua Serial

Ket noi may tinh vao cong RS485/RS232 cua NA111, mo terminal dung baud rate.

Cach vao AT Mode
# 1. Gui 3 dau cong (KHONG nhan Enter) +++ # 2. Trong vong 3 giay, gui lenh sau (co CR+LF) AT\r\n +OK <-- thiet bi phan hoi = da vao AT Mode # Kiem tra ket noi mang AT+LINKSTA\r\n +OK=CONNECTED / DISCONNECTED # Xem IP, Subnet, Gateway hien tai AT+WAN\r\n +OK=192.168.3.7,255.255.255.0,192.168.3.1 # Xem cau hinh cong Serial AT+UART\r\n +OK=115200,8,1,NONE # Xem che do TCP/UDP va IP/Port dich AT+SOCK\r\n # Khoi dong lai thiet bi AT+REBT\r\n # Factory Reset (can than!) AT+RESTORE\r\n # Thoat AT Mode AT+EXAT\r\n
Ma loi AT Command
-1Sai dinh dang lenh - kiem tra CR+LF o cuoi
-2Lenh khong ton tai - kiem tra chinh ta lenh
-4Tham so khong hop le - gia tri ngoai pham vi
